Canvas

  1. 获取 context(‘2d’)

  2. API

  • 绘图状态

  • 绘制命令

    • context.translate

      • 画布平移转换

    • context.save

      • 暂存状态

    • context.restore

      • 恢复状态
  1. 过程
  • 应当先处理数据，将数据内容与绘图指令建立映射关系，遍历数据，通过映射关系将数据内容的参数传给绘图指令

SVG

  1. 创建 svg 元素
  • document.createElementNS()
  1. 更改元素属性
  • element.setAttribute()
  1. 优点
  • 可以让图形的用户交互非常简单

WebGL

  1. 使用 WebGL 绘图，必须要和内存，GPU 打交道，控制图形输出的每一个细节

  2. 图形系统

  • 光栅

    • 构成图像的像素阵列

  • 帧缓存

    • 像素信息被存放于帧缓存中
  1. 绘制过程

  • 创建 WebGL 上下文

    • canvas.getContext(‘webgl’)

  • 创建 WebGL 程序

    • GLSL 编程语言

    • 两个着色器

      • 顶点着色器

        • 处理顶点的 GPU 代码，改变顶点的信息
        • 计算出需要着色的像素点

      • 片元着色器

        • 处理光栅化后的像素信息

        • 片元着色器对像素点着色的过程是并行的

          • 无论有多少个像素点，片元着色器都可以同时处理

  • 将数据存入缓冲区

    • 首先定义顶点

      • Float32Array 类型化数据

    • 将定义好的数据写入 WebGL 的缓冲区

      • createBuffer
      • bindBuffer
      • bufferData

  • 将缓冲区数据读取到 GPU

  • GPU 执行 WebGL 程序，输出结果

    • gl.drawArrays()

    • gl_FragColor

      • 改变当前像素点颜色
  1. 顶点和图元

  • 顶点

    • 几何图形的顶点

  • 图元

    • WebGL 可直接处理的图形单元
    • 由绘图模式决定
    • 点，线，三角形等

  1. 无论图形中有多少个像素点，着色器程序在 GPU 中会被同时执行多少次

  2. 顶点着色器

  • 作用

    • 通过 gl_position 设置顶点

    • 通过定义 varying 变量，向片元着色器传递数据

      • 通过给顶点映射不同的颜色，实现片元着色器渐变
